# Application-Development-with-Generative-AI
1.	Developed a Flask web application integrated with AI capabilities
2.	Utilized the ibm-watsonx-ai library to interact with advanced language models
3.	Implemented LangChain's JsonOutputParser for structured AI outputs
4.	Applied prompt engineering techniques for generating actionable JSON responses
5.	Compared and evaluated different language models, including Llama 3, Granite, and Mixtral


Setting up your development environment:

Step 1: Create your project directory
  mkdir genai_flask_app
  cd genai_flask_app

Step 2: Set up a Python virtual environment
  python -m venv venv
  activate venv

Step 3: Install the ibm-watsonx-ai library
  pip install ibm-watsonx-ai
  pip install Flask langchain-ibm langchain

Create all the other files in this folder:
1.	app.py
2.	config.py
3.	model.py
4.	llm_test.py
5. templates/index.html
